# ⚖️ Legal Document Generator

## 📌 Overview

Legal Document Generator is a web-based application that allows users to generate legal documents such as agreements and contracts automatically. It simplifies the process of document creation by providing a user-friendly interface.

## 🎯 Objective

The main objective of this project is to reduce manual effort in creating legal documents and provide a fast, efficient, and reliable solution.

## 💻 Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## 🚀 Features

* Dynamic legal document generation
* Simple and clean user interface
* Easy form-based input system
* Fast processing
* Secure data handling

## 🛠️ How to Run

1. Install XAMPP/WAMP
2. Move project to `htdocs`
3. Import database
4. Start Apache & MySQL
5. Open in browser
